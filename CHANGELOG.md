# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-29

### Added
- Initial release of `mdoc-cli`.
- Core commands: `convert`, `process`, `validate`, `lint`, `preview`, `config`, `version`.
- High-performance Markdown to HTML/JSON conversion.
- Cross-platform support for Windows, Linux, and macOS.
- NPM wrapper for easy distribution.
