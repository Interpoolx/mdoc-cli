# Product Requirements Document (PRD): MDoc CLI V2

## 1. Overview
The goal of MDoc CLI V2 is to provide the fastest markdown to PDF/HTML conversion experience for developers.

## 2. Target Audience
- Technical Writers
- Software Engineers
- DevOps Specialists

## 3. Key Features
- **Concurrent Processing**: Multi-threaded conversion.
- **Custom Templates**: Flexible layout options.
- **Theme Support**: Dark/Light modes out of the box.

## 4. User Stories
### AS A developer
I WANT to convert a directory of markdown files to HTML
SO THAT I can host my documentation easily.

## 5. Technical Constraints
> [!IMPORTANT]
> Must support Windows, macOS, and Linux without external dependencies.
